# Code of Conduct

## Our standard

This project aims to be a friendly, practical, and respectful place to collaborate.
Please communicate with patience, assume good intent, and focus feedback on the work.

Examples of expected behavior:

- being respectful in discussions and review comments;
- welcoming questions from new contributors;
- offering constructive, specific feedback;
- keeping technical disagreements professional.

Examples of unacceptable behavior:

- harassment, insults, or personal attacks;
- repeated bad-faith arguing or trolling;
- publishing private information without permission;
- discriminatory language or exclusionary behavior.

## Enforcement

Project maintainers may edit, remove, or reject comments, issues, pull requests, or
other contributions that violate this code of conduct.

## Scope

This applies to project spaces such as issues, pull requests, discussions, review
comments, and related community channels.
