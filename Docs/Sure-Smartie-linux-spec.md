## 1. Краткое резюме

Дисплей **SURE Electronics 20x4 LCD Smartie Asset** корректно работает в Linux через последовательный порт, но не является полностью совместимым с драйвером `MtxOrb` из `LCDproc`.

Практическая проверка показала:

- устройство определяется в Ubuntu 24 как **/dev/ttyUSB1**
- используется USB‑UART мост **CP210x**
- рабочая скорость обмена — **9600 baud**
- драйвер `LCDproc MtxOrb` работает некорректно
- при использовании **родного протокола устройства** дисплей работает корректно

Архитектурный вывод:

> проект должен использовать собственный драйвер дисплея

Цель проекта — создать Linux‑ориентированную систему вывода системных метрик на LCD‑дисплеи, аналогичную по концепции LCD Smartie.

## 2. Название проекта

**Sure-Smartie-linux**

---

## 3. Цель проекта

Разработать нативную утилиту Linux для дисплея **SURE Electronics 20x4 LCD Smartie Asset**, которая:

- работает напрямую через serial интерфейс
- использует родной протокол устройства
- выводит системные метрики
- поддерживает плагины источников данных
- может использоваться с различными типами индикаторов

---

## 4. Архитектурная идея

Система состоит из трёх уровней:

1. Core
2. Display Drivers
3. Providers / Plugins

Плагины **поставляют данные**, а ядро отвечает за форматирование и вывод.

## 5. Основные требования

### Функциональные

- работа с serial устройством
- вывод **4x20 символов**
- управление подсветкой
- управление контрастом
- управление яркостью
- циклическая смена экранов
- получение системных метрик
- конфигурация через файл
- расширяемость через плагины

### Графическое отображение

Поддержка псевдографики:

- bar‑графики
- шкалы загрузки
- индикаторы уровня
- progress bar
- custom LCD characters

Примеры:

GPU [████░░░░░░] 41%

- символьные LCD

- VFD
